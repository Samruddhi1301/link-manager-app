# 🔗 Link Manager & Task Tracker
https://samruddhi1301.github.io/link-manager-app/

A minimalist, frontend-based dashboard designed to centralize learning resources, documentation links, and custom checklists. Built with a focus on **Glassmorphism** and **Lofi aesthetics**.

## ✨ Features
- **Smart Task Splitting:** Add multiple tasks at once using commas, semicolons, or new lines.
- **Accordion View:** Clean dashboard that expands only when you need to see the details.
- **Granular Control:** Edit subject names/links and delete individual tasks without removing the whole topic.
- **Glassmorphism UI:** Soft, faint lofi color palette with frosted glass effects.
- **Zero Backend:** Uses `LocalStorage` so your data stays private on your browser. No sign-up required.

## 🛠️ Tech Stack
- Vanilla HTML5
- CSS3 (Flexbox, Grid, Backdrop-filter)
- Vanilla JavaScript (ES6+)
